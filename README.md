#CocosJsDev
> 在Sublime里开发cocos2dx-js的插件.


## 下载
> [github](https://github.com/FloydaGitHub/CocosJsDev)  
> [coding](https://coding.net/u/Floyda/p/CocosJsDev/git)  

+ git clone    
克隆到Sublime的Packages目录下.  
+ package control    
暂不支持.  


## 使用
+ 启动Local Server  
打开src目录下任意的js文件, 快捷键F1		
> 快捷键可自定义
